# `@canopy/arbor`

Arbor is Canopy's canonical generator and command suite. It owns application generation,
compilation, development hot reload, migrations, runtime roles, inspection, recovery, Drizzle
Studio, Cultivate knowledge, and Undergrowth operations.

```sh
pnpm dlx @canopy/arbor new MyApplication
cd my-application
pnpm install
pnpm dev
```

Run `arbor --help` for the complete command surface.
