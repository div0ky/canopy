# `@canopy/postgres-drizzle`

Canopy's first-party PostgreSQL transaction, durability, Eloquent-style model persistence, journal,
outbox, communications ledger, and cache adapter implemented with Drizzle.

```sh
pnpm add @canopy/postgres-drizzle
```

The package includes forward-only Canopy-owned migrations. Application models and domain code remain
independent from Drizzle types.
