# Ownership and Adoption

Doxa is independently owned and maintained by [div0ky](https://github.com/div0ky). Midtown Home
Improvements does not own or maintain the framework and is currently its sole supported production
consumer.

Framework development is performed independently, on personal time, in this separate repository.
Midtown applications install published Doxa packages as dependencies. Midtown application code,
schemas, business rules, credentials, infrastructure details, customer data, and private
documentation are not part of this repository or its npm artifacts.

Doxa prereleases are publicly downloadable and licensed under Apache-2.0. External use and forks are
permitted by that license, but public availability does not include a compatibility, support,
warranty, roadmap, or production-readiness commitment. The supported adoption program and
development roadmap remain closed.

This file records the project's ownership and adoption position. It is not a substitute for any
applicable employment, intellectual-property, or written inter-party agreement. Written confirmation
is recommended if ownership could otherwise be disputed.
